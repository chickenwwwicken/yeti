Useful for a single value
``` bash
git config --get <key>
```

Keys are in the format ```<section>.<key>```

Examples:
```user.name```
```webflyx.ceo```

#### Assignment
Use the ```--get``` flag to get the value of the ```webflyx.valuation``` from local Git configuration.
