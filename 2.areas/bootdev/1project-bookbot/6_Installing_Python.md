Installing Python on our local machines.

### For Linux

Verify if installed

``` bash
python3 --version

# Python 3.10.12
```

If not installed :

``` bash
sudo apt update
sudo apt -y python3
```

---

### For MacOS

Download Homebrew

``` bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

Open your `~/.zshrc`/`.bashrc` file and add this.

``` bash
brew install python3
```

---
### Check Installation

``` 
python3 --version
```
