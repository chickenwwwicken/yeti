Syntax for iterating directly over the items in a list without worrying about index numbers.
If you don't need the index number you can use the following syntax:

``` python
trees = ['oak', 'pine', 'maple']
for tree in trees:
	print(tree)

# prints:
# oak
# pine
# maple
```
