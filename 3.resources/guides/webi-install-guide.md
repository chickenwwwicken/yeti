### install webi
``` bash
curl -sS https://webi.sh/webi | sh
``` 
``` bash
# updating webi
webi webi
```
files/dirs created and /or modified with this install.
``` 
~/.config/envman/PATH.env
# Mac, Linux
~/.local/bin/webi
# Windows
~/.local/bin/webi.bat
~/.local/bin/webi-pwsh.ps1
```
